# If Statement

- Sama seperti di JavaScript, kita bisa melakukan if statement.
- Penulisan if statement sama dengan JavaScript.

## Kode If Statement

```js
const examValue = 90;

if (examValue > 80) {
  console.info("Good");
} else if (examValue > 60) {
  console.info("Not Bad");
} else {
  console.info("Try Again");
}
```
